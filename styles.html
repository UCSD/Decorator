A Note About Tables

In order to allow folks to continue using table-based layouts, the campus style will not mess with tables. Instead, if folks want a beautifully styled table for actual tabular data, they will add a class name "styled" to the table, like this:

Heading Heading Heading
Data 1 Data 2 Data 3
Data 4 Data 5 Data 6
Data 7 Data 8 Data 9

Note: the zebra striping is achieved using alternating "odd" and "even" class names on thetr elements.

<table class="styled">
    <tbody>
        <tr>
            <th>Heading</th>
            <th>Heading</th>
            <th>Heading</th>
        </tr>
        <tr class="odd">
            <td>Data 1</td>
            <td>Data 2</td>
            <td>Data 3</td>
        </tr>
        <tr class="even">
            <td>Data 4</td>
            <td>Data 5</td>
            <td>Data 6</td>
        </tr>
        <tr class="odd">
            <td>Data 7</td>
            <td>Data 8</td>
            <td>Data 9</td>
        </tr>
    </tbody>
</table>

The hard return ( br element) 
has a bottom margin of 0.5em. This helps distinguish it from a wrap or paragraph text. See how it helps group long links in the navigation to the left?

The page title and UC San Diego title at the top of each page are made with div elements instead of header elements. This helps promote better header hierarchy within the actual page content.

Drawers

The Drawer Template lets you organize content into "drawers" or expandable / collapsible subsections in the center of the page. Each drawer consists of a Drawer Title, where you enter a given section's title text and a Drawer Body where you input its corresponding content. This template is ideal for lengthy content (such as FAQs).
